My first 100 days of code challenge was dedicated to practicing algorithms and data structures. Looking back at this journey, though I have learnt quite a lot, I am not very happy with how it's disorganized. No roadmap or log to track my progress or a strict learning plan. Today, I learn little about recursion, the next day, I'm studying binary search and the next hashmap. It is hard to measure how far I have come.
